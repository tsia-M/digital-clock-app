# Digital Clock App

A responsive digital clock application built using HTML, CSS, and JavaScript.  
The application dynamically displays the current time, day, date, month, and AM/PM status in real time.

## Live Demo
https://tsia-m.github.io/digital-clock-app/

## Features
- Real-time clock updates using JavaScript
- Dynamic day, date, and month display
- Responsive mobile and desktop layouts
- Automatic AM/PM time convention handling
- Custom typography and styled UI
- Background image integration

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Concepts Practiced
- DOM manipulation
- JavaScript Date object
- setInterval timing functions
- Responsive web design
- Conditional rendering
- Dynamic UI updates

## Project Structure
- `index.html` → Application structure
- `styles.css` → Styling and responsive layout
- `script.js` → Clock logic and date/time functionality
